# label-2-frontend

> User interface for collector labelling system.

## Build Setup
Make sure you have following software installed in your system
* node v12.16.1
* npm v6.13.4

```bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
$ npm start
```

## Run Development Server
```
$ npm start
```
or you can use make
```
make
```

## Access
* host: `127.0.0.1`
* port: `3000`
